# ChainTrace User Getting Started

## 1. Select Domain

After login, choose your operational domain:

- Agriculture
- Pharmaceutical
- Food Safety
- E-commerce
- Warehouse IoT

The dashboard and product forms adapt based on this selection.

## 2. Register Product

1. Open **Register**
2. Complete the 5-step wizard:
   - Basic information
   - Product details
   - Certifications
   - Initial location
   - Review and submit
3. Submit to create product trace entry and generate QR

## 3. Add Checkpoints

Use checkpoint actions to record:

- Received
- Quality Check
- Processed
- Dispatched
- In Transit
- Delivered

Attach IoT values and notes as needed.

## 4. Verify Product

From the public verification screen:

1. Scan QR (or input product ID)
2. Review trust score and authenticity state
3. Inspect route checkpoints and evidence

## 5. Use Dashboard

- Monitor KPI cards
- Review timeline and alerts
- Explore AI insights
- View route map and 3D warehouse

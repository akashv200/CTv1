# FAQ

## Do consumers need an account to verify a product?
No. Verification works via public QR scan or Product ID lookup.

## Which blockchain is supported?
The default target is local Ganache (`http://127.0.0.1:7545`, chain ID `5777`) for zero-cost testing.

## Can I switch domains after setup?
Yes. Domain switching is available from the dashboard selector.

## Are IoT sensors required?
No. IoT can run in simulated mode for demo and academic evaluation.

## How are certificates stored?
Certificate file hashes are intended for IPFS + on-chain reference.
